# Moon Ridge SEO Optimization Skill

This skill defines the procedures and standards for maintaining peak search engine performance for the Moon Ridge Shopify store.

## Core Directives

### 1. Brand Identity & Authority
- **Brand Name:** Always use "Moon Ridge" or "Moon Ridge Company".
- **Legacy Reference:** Where appropriate for transition, use "(formerly Rafter M Hat Company)".
- **Title Prefix:** The standard SEO title structure is `Moon Ridge | Fayetteville's Hat and Heritage Store: [Page Name]`.

### 2. Meta-Data Standards
- **Page Titles:** 50–60 characters. Must be unique and keyword-rich.
- **Meta Descriptions:** 150–160 characters. Must include a call to action (e.g., "Shop now", "Book today").
- **Keywords:** Focus on "Heritage Brands", "Custom Hat Shaping", "Hand-Shaped Hats", "Fayetteville Hat Store", "Stetson", "Lucchese".

### 3. Image Optimization
- **Filename:** Use hyphens and keywords (e.g., `moon-ridge-custom-stetson-hat.jpg`).
- **Alt Text:** Mandatory for all images. Must describe the specific product and brand.

### 4. Technical SEO (Shopify Specific)
- **Schema.org:** Ensure `Organization`, `Product`, and `LocalBusiness` JSON-LD are correct.
- **Header Hierarchy:** H1 is reserved for the main page title. H2 for major sections. H3 for sub-sections.
- **Redirects:** Monitor for 404 errors during the rebrand transition from legacy Rafter M URLs.

## Operational Checklist (Audit Task)

When running this skill, perform the following checks:
1. [ ] Scan for residual "Rafter M" strings in meta tags and alt text.
2. [ ] Verify `meta-tags.liquid` is injecting the correct brand authority prefix.
3. [ ] Check `layout/theme.liquid` for valid Site Navigation and Organization schema.
4. [ ] Ensure all top-selling products have keyword-optimized descriptions.
